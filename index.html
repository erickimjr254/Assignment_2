Mathematical Projects

1. Easy Function: Quadratic Visualizer

This project explores the classic quadratic function f(x) = x² − 4x + 3. It includes root analysis, vertex computation, graph visualization, and explanations for beginners learning algebra.

Key Features:

  • Interactive parabola graph
  • Discriminant explanation
  • Vertex and roots calculator
2. Hard Function: Fourier Series Wave Synthesizer

This advanced project demonstrates how complex periodic functions can be approximated using Fourier series expansions. It lets users build waves by summing sine and cosine components.

Key Features:

  • Real-time Fourier reconstruction
  • Adjustable number of harmonics
  • Visualization of convergence and error
